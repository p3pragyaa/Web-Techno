Introduction to Class

A Java class file is a file containing Java bytecode that can be executed on the Java Virtual Machine. A Java class file is usually produced by a Java compiler from Java programming language source files containing Java classes.

Class Members

The components of a class, such as its instance variables or methods are called the members of a class or class members.

Introduction to Keyword

Java keywords are also known as reserved words. Keywords are particular words that act as a key to a code. These are predefined words by Java so they cannot be used as a variable or object name or class name.

Objects

A Java object is a member (also called an instance) of a Java class.

Constructors

In Java, a constructor is a block of codes similar to the method. It is called when an instance of the class is created.

Overloading

Overloading allows different methods to have the same name, but different signatures where the signature can differ by the number of input parameters or type of input parameters or both.
